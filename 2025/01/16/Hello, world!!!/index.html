搭建博客初体验

环境准备

Node.js

本次博客搭建基于Hexo框架。该框架需要在Node中运行，将markdaown文件转换成网页文件（static pages）。

git

版本控制程序（VCS, Version Control System）。当我们在本地更新完博客后，利用git将更新内容推送到github。

基本流程

1 建立新博客

这些信息会用于后续网页创建。因此，命令行更方便一些。

3 生成网页文件

将之前写的markdown文件转换成网页文件，并生成其他相关文件（CSS、.js）,生成的所有相关资源全部放在public文件夹

public文件夹就是后续要推到github上的所有资源，也是博客网站的所有资源

4 推送到github

第一次推送会要求填写个人github账号和密码，后续就不需要了。应该是git的credential manager组件记录了这些信息。
